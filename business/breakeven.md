# breakeven.md

## Unit Economics & Break-even Analysis for BuildFast

### Cost per Active User
- **Compute Costs**: $5/user/month
- **Storage Costs**: $2/user/month
- **Bandwidth Costs**: $1/user/month
- **Total Cost per Active User**: **$8/user/month**

### Pricing Tiers
| Tier Name         | Price ($/mo) | Features                                                                                       |
|-------------------|--------------|------------------------------------------------------------------------------------------------|
| **Starter**       | $15          | Basic templates, 1 project, community support, limited integrations                           |
| **Pro**           | $30          | All Starter features, unlimited projects, advanced templates, priority support, integrations   |
| **Enterprise**    | $75          | All Pro features, custom templates, dedicated account manager, API access, analytics dashboard |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$50 - $100** per user

### Lifetime Value (LTV) Estimate
- Average Subscription Duration: 24 months (based on market research)
- Average Revenue per User (ARPU): 
  - Starter: $15 * 24 = $360
  - Pro: $30 * 24 = $720
  - Enterprise: $75 * 24 = $1800
- Weighted LTV Estimate (assuming 60% Starter, 30% Pro, 10% Enterprise):
  - LTV = (0.6 * $360) + (0.3 * $720) + (0.1 * $1800) = **$552**

### Break-even Users Count
- Monthly Fixed Costs: $5,000 (including marketing, salaries, infrastructure)
- Contribution Margin per User (ARPU - Cost per Active User):
  - Starter: $15 - $8 = $7
  - Pro: $30 - $8 = $22
  - Enterprise: $75 - $8 = $67
- Break-even Users Count:
  - Starter: $5,000 / $7 ≈ **715 users**
  - Pro: $5,000 / $22 ≈ **227 users**
  - Enterprise: $5,000 / $67 ≈ **75 users**

### Path to $10K MRR
- Target Monthly Recurring Revenue (MRR): $10,000
- Pricing Tier Distribution:
  - **Scenario 1**: 300 Starter users → 300 * $15 = $4,500
  - **Scenario 2**: 200 Pro users → 200 * $30 = $6,000
  - **Scenario 3**: 50 Enterprise users → 50 * $75 = $3,750
  - **Total MRR**: $4,500 + $6,000 + $3,750 = **$14,250** (exceeds target)

- **Recommended Path**:
  - 200 Starter users ($3,000)
  - 150 Pro users ($4,500)
  - 50 Enterprise users ($3,750)
  - **Total MRR**: $11,250 (exceeds target)

This structure provides a clear path to achieving $10K MRR while balancing user acquisition across different tiers.
