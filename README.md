# Template Marketplace Project
This project provides a template marketplace where users can browse and import ready-made templates.

## Features
* Load templates from a JSON file
* Get templates by category
* Import a template by ID

## Usage
1. Create a `templates.json` file with the template data
2. Run the `template_marketplace.py` file to load the templates
3. Use the `get_templates` method to get templates by category
4. Use the `import_template` method to import a template by ID
